# SecureX일일 업무보고서 자동 생성기

이 폴더는 SecureX 기술지원팀의 일일 업무보고서 자동 생성 앱을 실습하기 위한 기본 화면입니다.

## 포함 파일

- `index.html`
- `style.css`
- `app.js`
- `README.md`
- `messenger_chat_sample.txt`
- `jira_task_sample.csv`

## 동작 방식

- 작성일을 선택합니다.
- 메신저 대화 내용과 JIRA CSV 데이터를 입력합니다.
- `샘플 데이터 불러오기 버튼`으로 참고 파일 내용을 가져올 수 있습니다.
- `로컬 샘플 보고서 생성`을 누르면 보고서 초안이 출력됩니다.
- `AI 보고서 생성`을 누르면 저장된 OpenAI API Key로 보고서를 생성합니다.
- `보고서 복사 버튼`으로 출력 결과를 클립보드에 복사할 수 있습니다.
- `TXT로 다운로드`로 요약표와 본문을 함께 텍스트 파일로 저장할 수 있습니다.
- `초기화 버튼`으로 입력값과 출력값을 비울 수 있습니다.
- `OpenAI API Key 입력 영역`에서 키를 저장하면 `AI 보고서 생성` 모드를 사용할 수 있습니다.

## 현재 단계

- 화면 구조, 기본 입력 동작, 샘플 데이터 불러오기, 요약표, 본문 보고서, 복사, TXT 다운로드, 로컬 샘플 보고서 생성, AI 보고서 생성까지 구현되어 있습니다.
- API Key는 `localStorage`의 `securex_api_key`에 저장됩니다.

## 실행 방법

이 프로젝트는 브라우저에서 열 수 있는 정적 페이지입니다.
다만 샘플 파일을 버튼으로 읽어오려면 같은 폴더를 웹서버로 열어 사용하는 편이 가장 안정적입니다.

예시:

```bash
python -m http.server 8000
```

그다음 브라우저에서 `http://localhost:8000`으로 접속합니다.
